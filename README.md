🚀 Sobre o Projeto
Desenvolvimento de uma aplicação web Full Stack para o gerenciamento de ordens e missões táticas em tempo real...

🛠️ Tecnologias Utilizadas
Frontend: React (Vite), JavaScript, CSS3
Backend: Java (Servlets, Jakarta EE), Servidor Apache Tomcat 10
Banco de Dados: PostgreSQL (pgAdmin)
Ferramentas: Maven, Postman
📡 Endpoints da API (Demonstração Postman)
1. Listar Missões (GET)
URL: http://localhost:8080/umbrella/api/missoes
Resposta (200 OK): Retorna a lista de diretrizes ativas em formato JSON.
2. Cadastrar Missão (POST)
URL: http://localhost:8080/umbrella/api/missoes
Body: x-www-form-urlencoded (titulo, descricao, prioridade)
